# Conversor de Monedas en Python

Este es un script sencillo que utiliza la API de [ExchangeRate-API](https://www.exchangerate-api.com/) para convertir dólares (USD) a otras divisas en tiempo real.

## 🚀 Instalación
1. Clona el repositorio.
2. Instala las dependencias: `pip install -r requirements.txt`
3. Ejecuta el script: `python conversor.py`

## 🛠️ Tecnologías
- Python 3
- API REST (JSON)
